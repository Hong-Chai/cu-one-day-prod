# Здравствуйте, уважаемые жюри!

Перед вами приложение для оценки педагогического качества лекций от экспертов индустрии **"Я препод?"**

Для того, чтобы можно было **увидеть** работу, установлена шуточная специализация и тема для лекции.

## Попробуйте по очереди вставить лекции в приложение:

Лекция 1 (не пройдет):
```
Согласно базовому правилу сложения, 2+2=4. Это одна из первых математических операций, изучаемая в школе, и она не требует доказательства 
```

Лекция 2 (пройдет):
```
Как понять, сколько будет 2 + 2. Что значит "прибавить". Когда мы пишем 2 + 2 это значит: к двум предметам добавить ещё два. 

Пример с предметами:
    Представим яблоки 🍎. 
    Сначала есть 🍎 🍎. Это 2 яблока. 
    Теперь добавим ещё 2: 🍎 🍎 + 🍎 🍎. 
    Соберём всё вместе: 🍎 🍎 🍎 🍎. 
    Считаем по одному: 1, 2, 3, 4. Получилось 4. 
Пример на пальцах. 
    Покажи ребёнку: Подними 2 пальца. Потом ещё 2 пальца. 
    Теперь посчитай все поднятые пальцы: 4. 
    Почему это всегда работает. Число 2 — это "два предмета".
    Когда мы делаем 2 + 2 мы просто объединяем две группы по 2. 

Итог: 2 + 2 = 4 потому что 2 предмета + ещё 2 предмета = 4 предмета
```
